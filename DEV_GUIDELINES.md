# Development Guidelines

## Tools

*   **Linting & Formatting**: This project uses [Ruff](https://docs.astral.sh/ruff/) for linting and formatting. Run `uv run ruff check .` to lint and `uv run ruff format .` to format your code.
*   **Dependency Management**: We use `uv` for dependency management.

## Coding Standards

*   **Type Hints**: Enforced Python type hints are required for all code.
*   **Naming Conventions**: All naming conventions must be in English.
*   **Branch Naming**:
    *   Features: `feat/`
    *   Bug Fixes: `fix/`
    *   Maintenance: `chore/`
